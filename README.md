# MintiqueX Static Website

Premium one-page static website for `mintiquex.com`.

## Files

- `index.html` - single-page landing site with embedded CSS
- `README.md` - deployment notes

## Sections

- Hero
- About
- Products
- Research Focus
- For Partners
- Contact

## Cloudflare Pages Deployment

1. Create a new Cloudflare Pages project.
2. Connect the repository or upload this folder directly.
3. Set the build command to blank.
4. Set the build output directory to `/` if deploying these files at the project root.
5. Deploy.

No npm, React, build tools, or paid external services are required.
